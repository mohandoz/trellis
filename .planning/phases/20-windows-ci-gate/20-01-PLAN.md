# Plan 20-01: conjure.ps1 PowerShell shim

## Goal

Create `conjure.ps1` at the repo root — a ≤30-line PowerShell shim that:
1. Locates Git Bash (`$env:ProgramFiles\Git\bin\bash.exe`) or WSL
2. Delegates `@args` passthrough to `cli/conjure`
3. Propagates exit codes with `exit $LASTEXITCODE`
4. Exits 2 with `Write-Error` if neither Git Bash nor WSL is found

## Acceptance Criteria

- File exists at `conjure.ps1` in repo root
- File is ≤30 non-blank lines
- `$ErrorActionPreference = 'Continue'` at top
- Git Bash discovery checks both `bin/bash.exe` and `usr/bin/bash.exe` paths
- WSL fallback uses correct `/mnt/<drive>` path conversion
- Error path: `Write-Error` + `exit 2`
- No hardcoded paths beyond standard `$env:ProgramFiles\Git` prefix

## Tasks

### Task 1 — Create conjure.ps1

Create `/Users/mohandoz/u01/innovate/conjure/conjure.ps1`:

```powershell
$ErrorActionPreference = 'Continue'

$gitBash = $null
foreach ($c in @(
    "$env:ProgramFiles\Git\bin\bash.exe",
    "$env:ProgramFiles\Git\usr\bin\bash.exe"
)) {
    if (Test-Path $c) { $gitBash = $c; break }
}

if ($null -ne $gitBash) {
    & $gitBash "$PSScriptRoot\cli\conjure" @args
    exit $LASTEXITCODE
}

if (Get-Command wsl -ErrorAction SilentlyContinue) {
    $drive = $PSScriptRoot.Substring(0, 1).ToLower()
    $rest  = $PSScriptRoot.Substring(2) -replace '\\', '/'
    wsl -- bash "/mnt/$drive$rest/cli/conjure" @args
    exit $LASTEXITCODE
}

Write-Error "conjure.ps1: Git Bash or WSL required. Install Git for Windows or enable WSL."
exit 2
```

## Verification

- `wc -l conjure.ps1` ≤ 30
- File contains `$ErrorActionPreference = 'Continue'`
- File contains `exit $LASTEXITCODE` (twice — one per bash path)
- File contains `exit 2` (error path)
