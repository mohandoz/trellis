# Plan 20-02: WIN-02 CI job + DEBT-01 ci-gate fix

## Goal

1. **WIN-02**: Add `windows-ps1-shim` job to `ci.yml` — `shell: pwsh`, tests `conjure.ps1 --version` exits 0 and exit code 2 propagates
2. **DEBT-01**: Fix `ci-gate` in `release.yml` — add empty-check guard + 5-attempt retry loop before the failure check

## Acceptance Criteria

### WIN-02
- `ci.yml` has `windows-ps1-shim` job with `runs-on: windows-latest`
- Job has at least one step using `shell: pwsh`
- Step 1 runs `conjure.ps1 --version` and asserts `$LASTEXITCODE -eq 0`
- Step 2 runs `conjure.ps1 init` (no target → exit 2) and asserts `$LASTEXITCODE -eq 2`

### DEBT-01
- `release.yml` ci-gate has retry loop: 5 attempts, 15s sleep
- Loop breaks when `total_count > 0`
- After all retries with `total_count == 0`: `echo "FAIL: no GitHub check-runs..."` + `exit 1`
- Existing failed-conclusion check preserved, uses stored `$response` variable

## Tasks

### Task 1 — Add windows-ps1-shim job to ci.yml

Append after the `windows-hook-wiring` job (end of file):

```yaml
  windows-ps1-shim:
    runs-on: windows-latest
    steps:
      - uses: actions/checkout@v4

      - name: Smoke-test conjure.ps1 --version (pwsh)
        shell: pwsh
        run: |
          & "$env:GITHUB_WORKSPACE\conjure.ps1" --version
          if ($LASTEXITCODE -ne 0) {
            Write-Error "conjure.ps1 --version exited $LASTEXITCODE (expected 0)"
            exit 1
          }
          Write-Host "OK: conjure.ps1 --version exit 0"

      - name: Confirm exit code 2 propagation (pwsh)
        shell: pwsh
        run: |
          & "$env:GITHUB_WORKSPACE\conjure.ps1" init
          if ($LASTEXITCODE -ne 2) {
            Write-Error "Expected exit 2 from conjure.ps1 init, got $LASTEXITCODE"
            exit 1
          }
          Write-Host "OK: exit code 2 propagated correctly"
```

### Task 2 — Fix ci-gate in release.yml

Replace lines 16-28 (the `run: |` block inside `Check CI status for tagged commit`) with:

```bash
echo "Checking CI checks for ${{ github.sha }}..."
max_attempts=5
attempt=0
response=""
while [ "$attempt" -lt "$max_attempts" ]; do
  attempt=$((attempt + 1))
  response=$(gh api "/repos/${{ github.repository }}/commits/${{ github.sha }}/check-runs")
  total=$(echo "$response" | jq '.total_count')
  if [ "$total" -gt 0 ]; then
    break
  fi
  if [ "$attempt" -ge "$max_attempts" ]; then
    echo "FAIL: no GitHub check-runs found for ${{ github.sha }} after $max_attempts attempts"
    exit 1
  fi
  echo "Waiting for check-runs to propagate (attempt $attempt/$max_attempts)..."
  sleep 15
done
result=$(echo "$response" | jq -r \
  '.check_runs[] | select(.name != "Release") | {name: .name, conclusion: .conclusion}')
failed=$(echo "$result" | jq -r \
  'select(.conclusion == "failure" or .conclusion == "timed_out" or
          .conclusion == "cancelled" or .conclusion == "action_required") | .name')
if [ -n "$failed" ]; then
  echo "FAIL: the following checks did not pass:"
  echo "$failed"
  exit 1
fi
echo "OK: all required checks passed"
```
