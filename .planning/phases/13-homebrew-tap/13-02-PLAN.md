<objective>
Add the mislav/bump-homebrew-formula-action@v3 step to .github/workflows/release.yml
so that every tagged release automatically updates the SHA256 in the
mohandoz/homebrew-conjure tap repo. This is the automation backbone of BREW-04.

Purpose: Without this step, every new release of Conjure requires manually computing
the tarball SHA256 and updating Formula/conjure.rb in the tap repo. The bump action
makes the entire update atomic: tag push → GitHub release → bump action fetches tarball,
computes SHA256, pushes updated formula to tap repo.

Output: .github/workflows/release.yml with one new step appended after "Create release".
This plan depends on 13-01 because the bump action targets Formula/conjure.rb in the
tap repo, which this repo's Formula/conjure.rb is the authoritative template for.
</objective>

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| GitHub Actions runner → HOMEBREW_TAP_GITHUB_TOKEN secret | Secret injected as env var; action reads COMMITTER_TOKEN; never echoed |
| bump-homebrew-formula-action → mohandoz/homebrew-conjure | Action fetches tarball over HTTPS, computes SHA256, pushes commit to tap repo using PAT |
| github.ref_name → download-url | Tag name flows into the tarball URL; only fires on v* push events per existing trigger |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-13-03 | Tampering | mislav/bump-homebrew-formula-action@v3 supply chain | accept | Using @v3 tag per D-07; Phase 15 will pin to exact commit SHA. Risk window: between tag move and SHA pin. |
| T-13-04 | Information Disclosure | HOMEBREW_TAP_GITHUB_TOKEN in action logs | mitigate | Token passed as env var (COMMITTER_TOKEN) to the action binary, not as a shell arg; GitHub Actions masks secrets by default in log output |
| T-13-05 | Tampering | download-url tarball substitution | mitigate | Homebrew verifies SHA256 of fetched tarball against formula; bump action computes SHA256 from same tarball URL before writing formula |
| T-13-SC | Tampering | npm/pip/cargo installs | accept | No package manager installs in this plan; the GitHub Action runs its own bundled Node.js binary |
</threat_model>
