    Add the mutate_archive function to lib/mutate.sh, inserting it between the mutate_rm function (line 77) and the mutate_summary function (line 84). Do not modify any existing function.

    Function signature: mutate_archive <src_abs> <archive_root>

    Implementation per D-11/D-12/D-13 and PATTERNS.md Pattern 5:

    1. DRY_RUN guard at top (identical to all other mutate_* functions): if DRY_RUN=1, echo "[dry-run] would archive ${src} → ${archive_root}/...", increment CONJURE_DRY_MUTATION_COUNT, return 0.

    2. Derive mirror path (D-12): rel="${src#/}" strips leading slash so that an absolute path /a/b/c.md becomes a/b/c.md. dest="${archive_root}/${rel}". This produces a path-preserving layout.

    3. mkdir -p "$(dirname "${dest}")" to ensure the parent dirs exist.

    4. cp -a "${src}" "${dest}" to copy preserving permissions and symlinks. If cp fails, return 1 (do not proceed to rm).

    5. Cross-platform sha256 (PATTERNS.md cross-platform sha256 pattern + Pitfall 4): check command -v sha256sum first; if found use sha256sum; else check command -v shasum; if found use shasum -a 256; else echo error to stderr and return 1 (abort — never rm without verify). Extract only the hash portion (cut -d' ' -f1).

    6. Compare src_hash and dest_hash. If they differ: print "[mutate_archive] ABORT: sha256 mismatch for ${src} — original preserved" to stderr; rm -f "${dest}" (clean up the incomplete dest); return 1. NEVER call rm on src if hashes do not match.

    7. rm -f "${src}" (only reached if hashes match — D-13 guarantee).

    8. Append ledger entry: printf '%s\t%s\t%s\t%s\n' "${src}" "${dest}" "${src_hash}" "$(date -u '+%Y-%m-%dT%H:%M:%SZ')" >> "${archive_root}/.archive-ledger"

    9. CONJURE_DRY_MUTATION_COUNT increment (one increment in live mode, at the end, counting the entire move as one mutation — per PATTERNS.md shared patterns section).
