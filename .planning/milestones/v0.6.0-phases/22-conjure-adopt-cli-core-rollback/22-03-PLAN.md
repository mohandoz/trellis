## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| manifest restructure_steps[] (skill-authored, untrusted) → mutate_* execution | `--apply-step` executes ops a (future) skill wrote; src/dest paths and op types are untrusted input that could traverse out of staging/target |
| .conjure-adopt-state created[]/mutated[] → rollback mutate_rm | rollback deletes paths from created[]; a poisoned created[] could delete unintended files |
| /dev/tty recovery input → destructive branch | the recovery prompt routes to rollback (destructive-adjacent); must require a conscious choice (no default) |
| staging/ content (skill-written) → CLAUDE.md/SKILL.md write | proposed content lands in staging before being written to project files; this is the pre-write inspection point RESTR-05 (Phase 23) builds on |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-22-09 | Tampering | path traversal via apply-step src/dest (`..`, absolute escape) | mitigate | Validate write `src` resolves UNDER `.conjure-adopt-state/staging/` and reject `..`; reject `dest` traversal before write; archive relies on mutate_archive's existing absolute/`..` rejection (lib lines 96-102) — exit 2 on any failure (D-07, RESEARCH Open Q3) |
| T-22-10 | Tampering | arbitrary write via staging-path content ref (D-07) | mitigate | `write` op content comes only from a real file under staging/; mutate_write is the single write path; no inline manifest content (D-07 rejects inline) |
| T-22-11 | Tampering | malformed/injected manifest JSON executing an unintended op | mitigate | jq parse-check + required-field check (id/op/status) + op-type allowlist {write,archive,extract}; reject with exit 2, never execute; all jq writes use `--argjson` (no shell string interpolation into JSON) |
| T-22-12 | Elevation of Privilege | rollback deleting files outside created[] | mitigate | mutate_rm only iterates state.json created[] (D-02 guarantees conjure's own dirs are excluded); sha256-verify of mutated[] catches any unrecorded mutation → exit 2 |
